# AWS — Manual OpenVPN setup

> 🚧 Work in progress.

A click-by-click guide to building the same OpenVPN server by hand — VPC, subnets, internet/NAT
gateways, route tables, security group, EC2, then installing and configuring OpenVPN — using only
the AWS Console and a Linux shell (no Terraform or Ansible).

For the automated version, see [`../iac/`](../iac/).
